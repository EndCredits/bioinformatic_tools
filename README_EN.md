# Custom Bioinformatic tools

Reusable Bioinformatics Tools Self-Implemented in Collaboration with AI.

# How to use

Add `./bin` directory to your PATH. And configure tools in your workspace if necessary.

# Now supports

- Visualization of transcription factor binding sites: `./visualize_tfbs`
